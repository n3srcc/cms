# CMS
 # Requirements
 - Node 18+
 - MongoDB +

## Stack
- Express
- React

# Setup
  ## webfront
  - client/.env
  ```
    REACT_APP_API_URL=http://localhost:8000
  ```
## Run
 ### api
  ```
    cd server && npm install
    npm run start
  ```
  ### client
  ```
    cd client && npm install
    npm run start
  ```
