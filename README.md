# web-movies
 # Requirements
 - Node 18+
 - MongoDB +
 - ReactJS 18+

## Stack
- Express 14
- FastApi 0.110.0

# Setup
  ## webfront
  - client/.env
  ```
    REACT_APP_API_URL=http://localhost:8000
  ```
## Run
 ### api
  ```
    cd server && npm install
    npm run start
  ```
  ### client
  ```
    cd client && npm install
    npm run start
  ```
